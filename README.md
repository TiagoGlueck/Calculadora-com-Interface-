# Calculadora com Interface
Calculadora com Interface usando Python

## Atualizações

- Implementação de uma estrutura com interface, controller e lógica

## Atualizações futuras

## Arquitetura
O projeto segue uma separação em três camadas:
- Interface (UI)
- Controller (controle das ações)
- Lógica (operações da calculadora)
  
  
## Visão Geral
Esta é uma calculadora gráfica simples desenvolvida em Python usando CustomTkinter. Ela permite operações básicas como soma, subtração, multiplicação e divisão, acumulando resultados sequencialmente em um total.

## Funcionalidades
Entrada de números via campo de texto.

Botões para somar (+), subtrair (-), multiplicar (*), dividir (/) e limpar o total.

Exibe o total atual e mensagens de erro (ex: divisão por zero ou valor inválido).

## Requisitos
Python 3.x.
Biblioteca CustomTkinter: instale com pip install customtkinter
